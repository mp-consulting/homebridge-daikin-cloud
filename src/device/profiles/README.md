# Device Profiles

Device profile definitions for identifying Daikin device types.

## Purpose

Profiles define patterns for matching Daikin devices to their appropriate accessory types based on:
- Device model information
- Available management points
- Device characteristics

## Device Types

- **Air Conditioning** - Standard Daikin AC units (dx4, dx23, etc.)
- **Altherma** - Heat pump systems with optional hot water tank
